
## 证据链检查清单
调试任何流程问题前，必须按顺序验证：
- [ ] 组件是否在运行？`ps aux | grep <name>`
- [ ] 配置是否正确加载？
- [ ] 输入是否到达？
- [ ] 处理逻辑是否执行？（加日志确认）
- [ ] 输出是否正确产生？
- [ ] 下游是否收到？

## Shell 命令前置检查
```bash
[ -d <path> ] || echo "目录不存在"
[ -f <file> ] || echo "文件不存在"  
which <cmd> || echo "命令不存在"
```

- `drwxr-xr-x` = 子目录存在（**看这一行**）
